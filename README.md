# Atom-Engine

Hice un fork del motor original y le metí mano para que funcione con OpenGL en vez de DirectX. También le arreglé varias cosas y mejoré otras. Quedó bastante piola lol

  Cambios principales

    Reemplacé DirectX por OpenGL - Ahora es cross-platform (probado en Linux, pero debería andar en Windows/macOS también)

    Optimizaciones de rendimiento - Algunas partes del código estaban medio lentas y las arreglé

    Código más limpio - Refactoricé partes que eran un lío

    Build system actualizado - Más fácil de compilar
